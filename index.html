Organize os atendimentos em um único lugar.

Use este site para registrar atendimentos, alimentar sua base de dados e facilitar a criação de dashboards e relatórios mensais.

Registrar novo atendimento Ver dashboard

O que este sistema registra?

Data, horário, canal, instituição, cargo, tipo de atendimento, segmento, programa, tecnologia envolvida, aula ou atividade e descrição do atendimento.

1. Registro

O colaborador preenche o formulário com os dados do atendimento realizado.

2. Banco de dados

Os dados podem ser enviados para uma planilha Excel Online por meio do Power Automate.

3. Relatórios

Com os dados organizados, você pode gerar dashboards no Excel, Power BI ou no próprio site.
